# Agentic JS Deobfuscator

This project aims to use agentic techniques to automatically deobfuscate JavaScript code, making it easier to analyze and understand obfuscated scripts. It leverages autonomous agents to iteratively analyze, transform, and simplify obfuscated JavaScript, ultimately producing a more readable and analyzable version of the original code.

It is served simply through a website built with HTML, CSS, and JavaScript, providing a user interface where users can input obfuscated JavaScript files and receive the deobfuscated output generated by the agentic deobfuscation process.
